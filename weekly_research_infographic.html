この事例のポイント
  • Chrome × AppleScriptでログイン済みのページを自動読み取り
  • 物販関連の30以上のキーワードで自動フィルタリング
  • 毎週コマンド1行で最新の活用事例を収集できる
  • まとめページから元の投稿に検索ヒント付きで飛べる
  • GitHub Pagesで公開 → チームメンバーにURLを共有するだけ

自分でも使いたい人へ（セットアップ手順）

  1. Claude Code をインストール
  2. ~/.claude/commands/ フォルダに weekly-research.md を保存
  3. Chrome → 表示 → デベロッパー → 「Apple EventsからのJavaScriptを許可」をON
  4. Chromeでリベシティにログインした状態で /user:weekly-research を実行
