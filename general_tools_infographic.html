いのっちのClaude Code活用事例 -- eBay/Carousellせどり

リベシティ発 -- 4つのツールの仕組みと作り方
1

AI戦略会議チーム

Claude Projects

Claude上にCEO・eBay専門家・経理・リスク管理・AI効率化・投資家・リアリストの7役のペルソナを設定し、事業の意思決定をAIチームで壁打ちする。

フロー
議題を投げる
→
7人のAIが各役割の視点でコメント
→
多角的な意見が出揃う
→
意思決定
作り方
  • Claude Projectsに7つのペルソナを設定したSystem Promptを作成
  • 各ペルソナに役割・性格・意見の方向性を定義
  • 「この議題について全員の意見を出して」と指示するだけ
  • CLAUDE.mdやProject instructionsに書くのがベスト
2

出品URLまとめ収集（Chrome拡張）

Chrome拡張 GAS スプレッドシート Claude Code

1つのブラウザウィンドウで開いている複数タブのURLをまとめて、スプレッドシートの出品リストシートに自動保存。

フロー
複数タブで商品ページを開く
→
Chrome拡張のボタンをクリック
→
全タブのURL+タイトルを取得
→
GAS WebアプリにPOST
→
スプシに一括保存
作り方
  • Chrome拡張のpopup.jsでchrome.tabs.query()により全タブ情報を取得
  • GASのdoPost()でスプシに書き込み
  • manifest.json(v3)でtabs権限を設定
  • Claude Codeでコード生成
3

よく使うスプレッドシート一覧をすぐ表示（Chrome拡張）

Chrome拡張 Claude Code

ツールバーのアイコンをクリックするだけで、よく使うスプレッドシートの一覧が表示され、ワンクリックで開ける。

フロー
ツールバーのアイコンをクリック
→
ポップアップにスプシ一覧が表示
→
クリックで該当スプシを新しいタブで開く
作り方
  • Chrome拡張のpopup.htmlにスプシ名とURLのリストをハードコード
  • シンプルなHTMLとCSSだけで構成
  • 追加・変更は「○○を追加して」とClaude Codeに言うだけ
  • ファイルはGoogle Driveに保存 → 複数PCで共有
4

Mac mini ストレージ整理

Claude Code Mac

Claude Codeにストレージの分析と整理を依頼し、1.2GB → 75GBまで空き容量を回復。

フロー
Claude Codeに「ストレージ整理して」と依頼
→
ディスク使用量を分析
→
バックアップ・写真をSSDへ移動提案
→
VM・キャッシュ・ログ類を削除
→
+74GB回復
作り方
  • Claude Codeに「Macのストレージを分析して、不要なファイルを特定して整理して」と依頼
  • du -shやfindコマンドで大容量ファイルを特定
  • 移動・削除の提案を確認して承認・実行
  • プログラミング不要、Claude Codeとの対話だけで完結
