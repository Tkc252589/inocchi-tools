基礎知識

Claudeには3つ＋αのモードがある

役割がかなり違うので、最初に整理しておこう

🧠 Chat

相談役。質問・壁打ち・文章作成。作業は自分でやる。

考える

⚡ Cowork

相談＋実際の作業まで進めてくれる。初心者に最おすすめ！

進める

🔧 Code

Coworkより高機能。複雑な業務・自動化に強い。中〜上級者向き。

深く触る

💡 ＋α の使い方

Claude in Chrome：ブラウザ拡張機能。開いているページをそのまま読んで作業できる
Terminal（CLI）版：コマンドラインから使う。自動化・複数工程の処理に最強

まずCoworkから → 慣れたらCode → 極めたいならTerminal

基礎知識

5つのモード 全比較

モード 使いやすさ 向いている人 主なできること
Chat ★★★ 高 相談・質問したい人 壁打ち・文章生成・調査
Cowork ★★★ 高 初心者・まず試したい人 ファイル整理・情報収集・実務サポート
Code（デスクトップ） ★★ 中 複雑な仕事を任せたい人 業務自動化・スキル登録・エージェント構築
Code（CLI） ★ 低 上級者・自動化を極めたい人 複数工程の自動化・PC作業全体
Claude in Chrome ★★★ 高 ブラウザ情報を使いたい人 ページ要約・情報抽出・ブラウザ操作
今日の参加者は「触ってみた人〜使いこなし中の人」まで様々。どのモードでもOK！

設定

最初にやる設定（5分で完了）

強力なツールだからこそ設定が大事。特に🚨は必ず確認！

🚨

絶対にオフにしてほしい設定が2つあります

オンにすると意図しないファイル削除・危険な操作が確認なしに実行される可能性があります

❌ 絶対 OFF

❌

バイパス権限モードを許可

場所：設定 ＞ Claude Code｜危険なコマンドを確認なしに実行してしまう

❌

すべてのブラウザアクションを許可

場所：デスクトップアプリ ＞ 一般｜Claudeがブラウザを自由に操作できてしまう

❌

ロケーションメタデータ ／ Claudeの改善にご協力

場所：設定 ＞ プライバシー｜位置情報・会話内容がAnthropicに送信される

✅ ON にすること

✅

コード実行

場所：設定 ＞ 機能｜これがないとClaude Codeが本来の力を発揮できない

✅

Claude in Chrome（コネクタ）

場所：設定 ＞ コネクタ｜「Webページを見て調べてきて」が使えるようになる

注意点

最初に知っておく注意点

⚠️ 本番データで最初から試さない

→ テスト用のサンプルデータで練習してから

⚠️ 大事なファイルはバックアップしてから

→ コピーを作ってから渡すのが安心

⚠️ 「なんでも許可」設定はしない

→ AIが何をしているか確認できる状態を保つ

⚠️ アウトプットは必ず確認する

→ AIのアウトプットは100%正しくないこともある

😊 AIは優秀だけど、ミスもする。確認の習慣が大事！
