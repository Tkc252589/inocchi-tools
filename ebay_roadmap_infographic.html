商品が売れたらスプシの在庫数を自動で減算。在庫0になったら他プラットフォームの出品も自動終了。売上データもスプシに自動記録。

作り方
  • eBay Notification APIでSold通知を受け取るWebhookをGASで作成
  • GASのdoPost()で通知を受信 → ItemIDからスプシの該当行を特定
  • 在庫数列を-1して、0になったらEndItem APIで出品終了
  • 売上データ（価格・日時・バイヤー）を売上シートに自動追記

eBay APIから売上を取得し、商品ごとの利益（売値 - 仕入値 - 送料 - 手数料）を自動計算。月次/週次の推移グラフも自動生成。

作り方

相場チェック & 提案 & 自動価格変更

同カテゴリ・同ブランドの他セラー価格を定期チェック。相場より高すぎ/安すぎを検知して推奨価格を提案。承認後に自動変更。
🔍 Browse API競合価格を取得
→
🤖 AI分析相場 vs 自分の価格
乖離率を計算
→
💡 提案表示スプシに推奨価格
＆理由を表示
→
✅ 承認→変更ReviseItem APIで
価格を自動更新
作り方
  • Browse API - searchで同キーワード・同カテゴリの出品を検索→価格リストを取得
  • 中央値・平均値と自分の価格を比較→乖離率を算出
  • 推奨価格をスプシの「提案」列に表示。色付きで高すぎ(赤)・安すぎ(青)を警告
  • スプシの「承認」列にチェック→GASがReviseItem APIで価格を一括変更
  • 週1回のGASトリガーで定期実行
Browse API - search Trading API - ReviseItem GAS トリガー 週1回自動実行
