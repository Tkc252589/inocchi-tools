AI自動生成の仕組み
🏷
SEOタイトル生成
  • 日本語の商品名+ブランド+写真を入力
  • ChatGPT Visionで写真を分析
  • 70〜80文字の英語SEOタイトル生成
  • SKUは含めない方針
📋
Item Specifics生成
  • Brand, Type, Color, Material等を自動判定
  • 写真からも情報を補完（Vision API）
  • 日本語混入チェック+自動再翻訳
  • 不明な項目はNAに設定
📝
HTML説明文生成
  • 商品スペック表（ブランド・素材・色等）
  • サイズ: H x W x D + inch変換 + Approx.表示

  • 状態ランク: 色付きハイライト表示
  • 関税・送料・返品ポリシー注記

スプレッドシート列マップ
📝 山田さん入力エリア (B〜X列)
B = SKU
C = 出品日 自動
D = 商品名（日本語）
F = ブランド   G = タイプ
I = サイズ (H x W x D)
N = 仕様 (ストラップ/ハンドル)
O = 状態総合   P = 外側   Q = 内側
V = Drive URL 自動
💰 価格・出品エリア (Y〜AN列)
Y = eBay Item ID 自動
AE = 販売価格 (USD)
AK = 英語タイトル AI
AL = 現在価格 (eBay同期)
AM = 変更価格 → eBay反映
AN = US送料 → eBay反映
🤖 AI生成エリア (AO〜AQ列)
AO = HTML説明文 AI
AP = Item Specifics JSON AI
AQ = Condition Description AI
💡
まだできること: 次の効率化提案
優先度 HIGH
1. 在庫連動 & 売切自動終了
商品が売れた時にeBay通知を検知 → スプレッドシートの在庫数を自動減算。 在庫0で他出品も自動終了。売上データもスプシに自動記録。
📊 効果: 売れた後の手動作業ゼロ + 在庫差異トラブル防止
優先度 HIGH
2. 売上・利益ダッシュボード自動生成
eBay APIから売上取得 → 商品ごとの利益（売値 - 仕入値 - 送料 - 手数料）を自動計算。 月次/週次の売上推移、ブランド別売上ランキングをスプシに自動生成。
📈 効果: 経営判断のスピードUP + どのブランドが稼げるか可視化
優先度 HIGH
3. 自動価格調整（競合分析）
同カテゴリ・同ブランドの他セラー価格をeBay Browse APIで定期チェック。 相場より高すぎ/安すぎを検知してアラート表示。「推奨価格」をAM列に自動提案。
💵 効果: 適正価格維持 → 回転率UP + 利益率の最適化
優先度 MID
4. Promoted Listings自動管理
出品後に一定期間売れない商品を自動でPromoted Listings（広告）に登録。 売れ行きに応じて広告レート（%）を自動調整。ROIを自動計算。
🚀 効果: 滞留在庫の回転率向上 + 広告費の最適化
優先度 MID
5. Best Offer自動対応
バイヤーからのBest Offerを自動チェック。最低価格以上なら自動Accept、 以下ならカウンターオファーを自動送信。交渉履歴をスプシに記録。
⚡ 効果: オファー対応の即時化 → 成約率UP（バイヤーは待つと離脱する）
優先度 MID
6. 追跡番号自動登録 & 発送通知
発送時にスプシに追跡番号入力 → eBayに自動登録。 発送完了メールをバイヤーに自動送信。配達完了ステータスも自動記録。
📦 効果: 発送作業の手間削減 + Seller Rating向上
優先度 LOW
7. 出品テンプレートA/Bテスト
異なるタイトル・説明文パターンのクリック率を比較。 効果の高い表現を学習して次の出品に反映。
📊 効果: データに基づくリスティング最適化 → 長期的な売上向上
優先度 LOW
8. リターンリクエスト自動対応
返品リクエスト自動検知。理由に応じた定型対応 （部分返金/返品承認/拒否）を自動実行。返品率アラート表示。
🛡 効果: 返品対応の迅速化 + ケースエスカレーション防止
