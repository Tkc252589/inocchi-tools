Chrome拡張機能 3種

🔵 なごさん用（リサーチ）

  • メルカリの商品タブを自動検出
  • 商品情報（タイトル・価格・画像・セラー名）を抽出
  • 出品リストに一括保存
  • ボタン1つで一括転記も可能

🟢 オーナー用（直接転記）

  • メルカリ商品を即座にCarousell出品リストへ
  • ChatGPTで英語SEOタイトル自動生成
  • Carousell用テンプレート説明文も自動作成
  • 価格の自動換算

🔄 フォローリスト同期

  • メルカリのフォロー中アカウント・ショップを自動取得
  • 検索条件も含めてスプシに同期
  • ★注目マーカーで重要セラーをハイライト
  • 「もっと見る」も自動クリック

スプレッドシートの構成

列出品リスト（中間）出品して欲しいリスト（最終）

V—登録者マーカー 🟢🔵 + 日付

重複チェック

  • 出品リストに保存する際：既存URL（出品リスト + 転記先の両方）と照合
  • 一括転記する際：転記先に既にあるURLはスキップ
  • スキップされた行は 赤背景 + B列に「重複」と表示
  • 日付は入れない（重複と正常処理が一目で区別可能）

構築方法

1

Claude Code に要件を伝える

「メルカリの商品情報を自動で取得してスプシに送りたい」「ChatGPTで英語に翻訳したい」など、やりたいことを日本語で伝えるだけ

2

Chrome拡張 + GASを自動生成

manifest.json、background.js、popup.html/js、GASコードをすべてClaude Codeが自動生成。clasp CLIでデプロイまで自動化

3

Chrome拡張をロード

chrome://extensions → デベロッパーモード → パッケージ化されていない拡張機能を読み込む → フォルダを指定

4

改善のループ

「セラー名も取得したい」「V列にマーカーを入れたい」→ Claude Codeに伝えるだけで即修正・再デプロイ
